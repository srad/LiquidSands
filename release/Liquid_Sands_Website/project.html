What is Liquid Sands?

Liquid Sands is a round based strategy game which has been build as a project at the Goethe-University in Frankfurt.

The project has been build at the computer science department by students in the course Visual Computing.


Who made this project?

The game including this web site has been build by the course attendees: Wladimir, Bernd, Saman as their course project, all computer science students.

Also all game models and graphics has been build by the students with Blender and Adobe Photoshop, some textures has been taken from the jmonkey library.


Under which license is the project?

The game project consists of many different parts which must be handles separately.

  1. The game engine

    The game engine is jMonkeyEngine 3.0 which is under a BSD license, see here for details

  2. The 3d models

    All 3d models within the game has been build by Wladimir and Bernd and are not free for commecial use only for open source projects.
    Any other use must be granted explicitly by these creators.

  3. The textures

    The model textures are under the same license as the 3d models.

    Any other texture is part of the jMonkeyEngine 3.0 package and detials must be taken from the official jMonkeyEngine 3.0 license.
