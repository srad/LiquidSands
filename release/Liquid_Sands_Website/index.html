Welcome to our project home page.

Liquid Sands is a round based strategy game build on top of the free java 3d-game engine jMonkeyEngine 3.0.

Feel free to learn more about the game in our Game-Giude Section, which will cover the game details.

Our game source is freely available under a creative commons license and can be used freely for any non commercial purposes.
